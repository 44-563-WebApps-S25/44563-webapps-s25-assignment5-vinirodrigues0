Which row breaks first?

After resizing the browser window, it was observed that the second row (pet names) collapses into a vertical stack before the first row (letters). This happens because the first row uses the 'col-sm-2' class, which maintains its layout at smaller screen sizes. While the second row uses 'col-md', which stacks earlier when the screen width decreases.
